# MS5637 Temperature Compensated Barometer for Arduino

This library provides support for the MS5637 temperature-compensated barometer in Arduino. This library currently only works on Teensy 3.x. Feel free to do a pull request to add support for 8bit AVR boards.

## Usage

Copy this directory in your Arduino/libraries directory and find the "TempAndBarometer" example under the MS5637-Barometer
